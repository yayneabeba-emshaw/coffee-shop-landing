#Aroma Cafe Landing Page
A responsive landing page for a fictional coffee shop built with HTML and CSS.
## Texhnologies used

- HTML5
- CSS3
- Git & GitHub

## Features
- responsive design (mobile, tablet, desktop)
- fixed navigation
- Interactive menu grid
- Contact form
- Smooth scrolling

## What I Learned
- How to structure a responsive landing page layout using HTML and CSS
- Using CSS Grid and Flexbox for modern layouts
- Implimenting fixed navigation,footer and smooth scrolling
- Organizing project files and maintaning clean code
- Hosting projects with GitHUB pages
